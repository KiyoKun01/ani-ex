# Contribution Guidelines

Thank you for your interest in contributing to ANI-EX! 

Since this project is still in its early stages, it might have some bugs and is not entirely stable yet. We appreciate anyone who wants to contribute, whether it's fixing bugs, adding new features, or improving the documentation.

## How to Contribute

1. **Fork** the repository
2. **Create** a feature branch: `git checkout -b feature/my-feature`
3. **Commit** your changes: `git commit -m "feat: add my feature"`
4. **Push** to your fork: `git push origin feature/my-feature`
5. **Open** a Pull Request

Every contribution, no matter how small, is very much appreciated!
